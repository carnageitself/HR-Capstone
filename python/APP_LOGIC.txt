================================================================================
HOW THE APP WORKS
================================================================================

WHAT WE DID:
1. Generated taxonomy (6 categories A-F) using Claude
2. Generated 1000+ awards with Claude, categorized them
3. Generated 60 employees as base data
4. Built web dashboard on top of this data

WHAT THE APP DOES:
• Shows 1000+ awards in a dashboard
• Search/filter employees by name or department
• View award details with category and reasoning
• Analytics charts for insights
• Admin panel to add new data

================================================================================
HOW IT WORKS
================================================================================

FRONTEND (what you see):
├── frontend/index.html      Dashboard UI
├── frontend/app.js          Makes it work
└── frontend/styles.css      Styling

BACKEND (server):
└── backend/app.py           Flask API

DATA (storage):
├── data/awards.csv          1000+ awards (already categorized)
├── data/employees.csv       60 employees
└── data/award_categories.csv 6 categories (A-F)

Flow: Browser → app.js calls → app.py reads CSV → displays data

================================================================================
TAXONOMY - 6 CATEGORIES (A-F)
================================================================================

A = Leadership & Management      (leading teams, mentoring)
B = Innovation & Business Impact (new ideas, business results)
C = Customer & Client Focus      (customer service, relationships)
D = Teamwork & Collaboration     (team support, collaboration)
E = Process & Operational        (efficiency, systems)
F = Learning & Development       (training, growth)

Every award is one of these 6 types.

================================================================================
IF YOU WANT TO GENERATE TAXONOMY FOR YOUR OWN DATA
================================================================================

File: backend/taxonomy_pipeline.py

This generates taxonomy efficiently:

3-PHASE METHOD (Best):
1. Phase 1: Claude seeds from 100 samples ($0.10)
2. Phase 2: Llama processes all 1000 locally ($0)
3. Phase 3: Claude finalizes ($0.10)
Total: $0.20 (vs $50 if Claude did all 1000)

SAMPLING METHOD (Simple):
1. Take 100 sample awards
2. Send to Claude: "What categories?"
3. Claude returns taxonomy
Cost: 1 API call

How to use:
  cd backend
  python taxonomy_pipeline.py
  [provide your CSV]
  [system generates output]
